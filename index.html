Import Soal dari File
Format File JS
Upload file .js dengan variabel PAKETS mengikuti struktur questions.js yang sudah ada. File baru akan di-merge dengan soal yang ada.

Contoh minimal: 
const PAKETS = [
  {
    id: 11,
    title: 'Paket 11 – Judul Paket',
    topic: 'Pancasila',
    pg: [
      {
        q: 'Teks pertanyaan PG?',
        options: ['A. Pilihan A','B. Pilihan B','C. Pilihan C','D. Pilihan D'],
        answer: 'A',
      },
    ],
    essay: {
      q: 'Teks pertanyaan essay?',
      rubrik: 'Rubrik penilaian...',
    },
  },
];
Format File DOCX
Buat file Word dengan format berikut untuk setiap soal. Pisahkan antar soal dengan baris kosong. 
PAKET: 1
TIPE: PG
PERTANYAAN: Teks pertanyaan di sini?
